Detailed Description

CDigFloat: a class for floating point numbers which handels numerical errors of the 64 bit floating point representation.
This class offers:

  • operations of floating points (+,-,/,*)
  • propagation of numerical errors (round-off errors ) for all operations performed on an instance
  • setting of user defined precision for comparing and printing.
